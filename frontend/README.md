# Sweet Shop Management System - Frontend

Modern React application for the Sweet Shop Management System.

## Features

- **Authentication**: Login and Register pages.
- **Dashboard**: View and search sweets.
- **Admin Panel**: Manage inventory (Add, Edit, Delete, Restock).
- **Responsive Design**: Built with Tailwind CSS.

## Setup

1.  **Install Dependencies**:
    ```bash
    cd frontend
    npm install
    ```

2.  **Run Development Server**:
    ```bash
    npm run dev
    ```

3.  **Build for Production**:
    ```bash
    npm run build
    ```

## Tech Stack
- React (Vite)
- Tailwind CSS
- Framer Motion
- Axios
